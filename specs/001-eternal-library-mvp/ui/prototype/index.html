技術架構

React ↔ Phaser 通訊

START_DRAW → DrawScene
DRAW_COMPLETE → React
REVEAL_CARD → CardRevealScene
START_CELEBRATION → CelebrationScene

懶載入策略

首頁 Bundle < 500KB，Phaser 3 (~400KB) 僅在使用者點擊「抽取解答」時載入。

響應式設計

支援桌面（1280px+）、平板（768px-1279px）、手機（< 768px）三種斷點。

完整互動流程

  1. 使用者在登入頁連接錢包
  2. 進入主頁面，查看 MGC 餘額和簽到狀態
  3. 完成每日簽到，獲得 5 MGC（動畫效果）
  4. 在文字框輸入問題，點擊「抽取解答」
  5. 懶載入 Phaser.js，顯示 DrawScene（卡牌飛入動畫）
  6. 切換到 CardRevealScene（持續光效）
  7. React 覆蓋層顯示答案文字和稀有度
  8. 點擊「鑄造 NFT」，執行 CelebrationScene（慶祝特效）
  9. 跳轉到收藏頁面，展示新鑄造的 NFT
