# T020 學習報告：Card 元件

**任務編號**：T020
**技術等級**：B 級（基礎 UI 元件）
**完成時間**：2025-12-16

## 核心功能

1. **可選標題**：支援 title prop 顯示卡片標題
2. **內容容器**：提供一致的內邊距和間距
3. **視覺樣式**：白色背景、圓角、邊框、陰影

## 實作重點

**簡潔設計**：
- 只有 40 行程式碼
- 清晰的結構分層（標題區 + 內容區）
- 完全可自訂樣式

**使用範例**：
```tsx
// 帶標題的卡片
<Card title="使用者資訊">
  <p>卡片內容...</p>
</Card>

// 純內容卡片
<Card>
  <p>無標題卡片</p>
</Card>
```
